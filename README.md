## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS for customization

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tripplanner.git
   cd tripplanner
   ```

2. Open `index.html` in your browser to view the website.

### Local Development Server

For a better development experience with all features working properly:

```bash
# Using Python
python -m http.server

# Using Node.js
npx serve
```

Then visit `http://localhost:8000` (Python) or `http://localhost:5000` (Node) in your browser.

## 🎨 Customization

### Modifying Content

1. Update text content in HTML files
2. Replace images in the `/images` directory
3. Modify package details in `/packages/packages.html`
4. Update service information in respective service directories

### Styling Changes

1. Main styles are in `/css/travlo.webflow.9d605c0ed.css`
2. Additional inline styles are present in individual HTML files
3. Custom fonts can be replaced in the `/fonts` directory

### Adding New Destinations/Services

1. Duplicate an existing destination/service page
2. Update content, images, and links
3. Add navigation links to the new page from relevant sections

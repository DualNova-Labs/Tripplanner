## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS for customization

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tripplanner.git
   cd tripplanner
   ```

2. Open `index.html` in your browser to view the website.

### Local Development Server

For a better development experience with all features working properly:

```bash
# Using Python
python -m http.server

# Using Node.js
npx serve
```

Then visit `http://localhost:8000` (Python) or `http://localhost:5000` (Node) in your browser.
